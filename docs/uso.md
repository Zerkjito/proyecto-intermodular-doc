# Uso de la aplicación

## Usuarios tipo

- Administrador
- Profesorado
- Alumnado

## Casos de uso

- Gestión de usuarios (tanto alumnos como profesores)
- Gestión de alumnos y organización de horarios y clases (profesorado)
- Gestión de clases reservadas (alumnado)
- Consultar disponibilidad
